## 📌 รายละเอียดโปรเจกต์
## Features

### สำหรับลูกค้า
- แสดงรายการเมนู (อาหาร / เครื่องดื่ม / ของหวาน)
- ค้นหาเมนูแบบ realtime (while typing)
- กรองเมนูตามประเภท (Category)
- ดูรายละเอียดสินค้า (ราคา / รูปภาพ / ตัวเลือก)
- เพิ่มตัวเลือก (เช่น ไข่, ความหวาน, ความเผ็ด)
- เพิ่มสินค้าเข้าสู่ตะกร้า
- แสดงจำนวนสินค้าในตะกร้าแบบ live
- คำนวณราคารวมอัตโนมัติ
- ใช้โค้ดส่วนลด (Promotion)
- ยืนยันคำสั่งซื้อ
  
### สำหรับร้านอาหาร
- ดูรายการคำสั่งซื้อ (Orders)
- จัดการเมนูอาหาร (เพิ่ม / แก้ไข / ลบ)
- จัดการหมวดหมู่ (Category)
- ดูยอดขาย (Sales Summary)
- ตรวจสอบสถานะคำสั่งซื้อ
